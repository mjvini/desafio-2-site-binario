CLIQUE| CLIQUE PARA COMEÇAR Bzzz!  Zumbidos curiosos, amiguinhos! Vamos falar sobre a base decimal, que é como falamos os números no dia a dia. Vocês já devem estar acostumados... Na base decimal, usamos dez algarismos: 0, 1, 2, 3, 4, 5, 6, 7, 8 e 9. Vamos ver o número 29, por exemplo... Nele o \"2\" está na casa das dezenas, e o "9" está na casa das unidades. Isso quer dizer que o número 29 é o mesmo que (2*10) + (9*1). Simples, né?... Agora vamos olhar para o número 99. Se somarmos 1 temos 100, certo? Mas por que será?... Na base decimal a casa mais à direita representa o 1 (a casa das unidades), e cada casa que adicionamos à esquerda é 10 vezes maior que a anterior... Em 99 temos o "9" na casa das dezenas e o "9" na casa das unidades, isso é o mesmo que (9*10) + (9*1), exatamente igual no 29. A nossa maior casa em 99 vale 10, a casa das dezenas, então adicionamos mais uma: essa vale 100!!... Agora fica fácil: 100 na base decimal é o mesmo que (1 * 100) + (0 * 10) + (0 * 1), ou seja, 100+0+0. Demais, não é mesmo?... Não é legal ver como os números do nosso dia a dia funcionam? Vamos entender mais um pouco sobre eles no balãozinho debaixo! Bzzz!

CLIQUE| CLIQUE PARA COMEÇAR Bzzz! Zumbidos beenários, amiguinhos! Vamos agora desbravar os segredos da base binária. Essa é uma linguagem diferente, mas igualmente fascinante. Cá entre nós, essa é a minha preferida! Na base binária, usamos apenas dois algarismos: 0 e 1. É como se fossem os "sim" e "não" dos números. Imaginem, cada número é formado apenas por esses dois dígitos! Aqui, a casa mais à direita ainda vale 1, mas diferente do sistema decimal, dobramos o valor de cada casa adicionada. Vejam: se esbarramos no número 100, a casa onde está o "0" à direita vale 1, o próximo "0" está na segunda casa, que vale 2, e o último "1" está na terceira, que vale 4. Traduzindo, o número 101 é (1 * 4) + (0 * 2) + (0 * 1). Fácil, né? No início pode parecer um pouco confuso, mas com o tempo a gente se acostuma. Agora, vamos explorar o 1101. Aqui, o "1" está nas primeira casa, o próximo "0" na segunda, o próximo "1" na terceira e o último "1" na quarta. Isso se traduz em (1 * 8) + (1 * 4) + (0 * 2) + (1 * 1), o mesmo que 13 em decimmal. Incrível, não acham? Bzzz! Incrível como os zeros e uns se combinam para criar todos esses números na base binária, não é mesmo? Nessa base conseguimos escrever qualquer número e é ela que o computador usa no dia a dia! Continuem zumbindo por aqui, por que na próxima seção vamos entender como esses números são usados para formar palavras! Bzzz!

CLIQUE| CLIQUE PARA COMEÇAR Bzzz! Como foi divertido aprender as bases com vocês! Aqui, vamos explorar como os computadores entendem a linguagem especial dos números binários. Preparem-se para uma viagem emocionante pelo universo dos zeros e uns! Vocês sabiam que os computadores falam binário? Pois é, é como se fosse a língua deles, e eu vou explicar como isso funciona. Imagine que cada interruptor em um computador pode estar ligado (número 1) ou desligado (número 0). Agora, nós agrupamos esses interruptores em conjuntos de oito. Chamamos eles de bytes. Quando você digita algo no teclado, o computador converte essas letras e números para a linguagem binária. Cada letra, número ou símbolo é representado por uma combinação única de zeros e uns. Por exemplo, a letra "A" pode ser representada por 01000001. Cada dígito (0 ou 1) nessa sequência é chamado de bit, e oito bits formam um byte. Assim, "A" é como um código binário secreto, conhecido pelo computador. Quando você salva um arquivo, o computador transforma todo o conteúdo, seja texto, imagem ou vídeo, em sequências de zeros e uns. E quando você abre novamente, ele traduz esses zeros e uns de volta para o que você estava vendo na tela. Isso é incrível, não é? Os computadores falam uma língua de zeros e uns, e é assim que conseguem processar informações tão rapidamente. Então, na próxima vez que estiverem mexendo no computador, lembrem-se: é tudo uma dança mágica de binários! Bzzz!
CLIQUE| CLIQUE PARA COMEÇAR Bzzz! Que legal, agora já sabemos que os computadores usam o sistema binário para armazenar as informações! Agora, vamos explorar a incrível tabela chamada ASCII, que é como uma enciclopédia que ajuda os computadores a entenderem e representarem letras, números e símbolos (chamamos eles de caracteres 😉). Vamos dar uma espiadinha nessa tabela mágica aí em baixo! A tabela ASCII é como um dicionário dos caracteres, onde cada letra, número ou símbolo possui um código único. Esses códigos são representados em binário, como os zeros e uns que já conhecemos. Cada caractere tem seu próprio endereço na tabela. Vimos no exemplo anterior que o código binário 01000001 representa a letra "A". Esse código está listado na tabela ASCII para garantir que todos os computadores entendam que "A" é 01000001 (e vice-versa). A tabela ASCII vai além das letras: ela inclui números, pontuações e até mesmo comandos especiais. Ela é como um dicionário universal que permite aos computadores e programas se comunicarem de maneira padronizada, para que todos falem a mesma língua. Então, da próxima vez que digitarem uma letra no teclado, saibam que o computador está consultando sua tabela ASCII para entender qual código binário representa aquela letra específica. Em resumo, a tabela ASCII é como um guia de tradução para os computadores, ajudando-os a interpretar e exibir os caracteres que nós, humanos, utilizamos em nossas comunicações digitais. Fascinante, não é mesmo? Dá uma olhada nela aí embaixo e pratique a tradução! Bzzz!
